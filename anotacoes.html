Descrição / Tag
Title
A tag title serve para colocar o titulo da page. <"title">
DIV
A tag div, serve como um container para separar as partes que desejar.
<"div">

IMG
A tag IMG serve para adcionar imagens quando necessário, nela não é necessário fechar tag com / barra.
<"img">

BR
A tag br, serve para pular uma linha, uma quebra de linha. Não é necessário fechar a tag com a barra /.
<"br">
