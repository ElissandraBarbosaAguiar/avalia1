HTML & CSS



Descrição / Tag
height="…"
Define uma altura para o elemento(No CSS)
DIV
A tag div, serve como um container para separar as partes que desejar.
<"div">

IMG
A tag IMG serve para adcionar imagens quando necessário, nela não é necessário fechar tag com / barra.
<"img">

BR
A tag br, serve para pular uma linha, uma quebra de linha. Não é necessário fechar a tag com a barra /.
<"br">


class=" Nomedaclasse"
Atribui uma classe ao elemento (uma classe pode ser utilizada para um ou mais elementos).


id="identificador"
Atribui um id ao elemento (um id deve ser único, ou seja atribuído a um único elemento).


style="..."
Permite incluir elementos CSS (estilos) dentro da tag.
lang="pt_br"
Define o idioma principal do elemento. Essa tag vai no inicio do código.


alt="…"
Define um texto alternativo e, por isso, é muito utilizado em imagens, auxilia nas práticas de SEO; Na descrisção de imagens.

hidden
Oculta o elemento <"hidden">
align="..."
Permite definir o padrão de alinhamento desse elemento, como por exemplo: right, center, left e justify. (No css)


width="..."
Define uma largura para o elemento(No css)
