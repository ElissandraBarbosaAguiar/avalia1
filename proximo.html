HTML & CSS



Descrição / Tag
Tag de texto
<"p"> Principal tag de texto, compõe um parágrafo.
SPAN
<"span"> Apesar de ter uma funcionalidade e características parecidas com os parágrafos, costumam ser utilizadas apenas para pequenas informações, como legendas de um formulário, legendas de uma imagem, etc.

PRE
<"pre"> Tag utilizada para representar texto pré-formatado. Muito utilizada para inserir códigos

B
<"b"> Transforma o conteúdo em negrito


I
<"i"> Transforma o conteúdo em itálico.


HR"
<"hr"/> Essa tag não necessita de fechamento, ela forma uma linha horizontal.


IMG
Essa tag não necessita de fechamento, serve para incluir uma imagem ao seu texto. A partir dessa tag, utilizamos o atributo src=".." onde deve ser digitado o local em que a imagem se encontra.
FORM
Para iniciar um formulário, incluimos as tags <"form"> e <"/form">.


INPUT TYPE TEXT
<"input type=”text”> Define um campo que receberá qualquer caractere.

INPUT TYPE EMAIL
<"input type=”email”> Define um campo que receberá caracteres e verificará se o mesmo consiste em um e-mail válido;>
INPUT
<"input type=”submit” value=”ENVIAR”> Define um botão que servirá para o envio do formulário. Dentro dele, podemos atribuir o value, que será o texto dentro do botão de envio.


STYLE E HEAD
A tag e <"/style"> deve ser incluída no <"head> do seu código HTML. Dentro dessa tag, é possível incluir todo o seu código CSS, ou seja, seu código de estilos.
